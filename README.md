# O que é Criptografia RSA?

RSA é um método de criptografar informações baseado na dificuldade de fatorar numeros primos grandes. Ela funciona com duas chaves, uma pública (feita para criptografar a informação desejada) e uma privada (para descriptografá-la). As formulas/calculos complexos envolvidos na criação das chaves torna esse método virtualmente impossivel de descobrir a chave privada por meio da chave pública
